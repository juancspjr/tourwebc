PASO 2: MODIFICAR FORMULARIO EN dist/index.html

Localizar: <form> o <form id="contactForm"> o similar

Cambiar el atributo action de:
- action="/api/contact" 
- action="http://localhost:..."
- action="fetch(...)"

Por:
- action="https://formsubmit.co/admin@riotripvibes.com"

Y agregar campos ocultos DENTRO del form:

<input type="hidden" name="_next" value="https://riotripvibes.com">
<input type="hidden" name="_subject" value="Nueva Reserva - Rio Trip Vibes">

ESTRUCTURA FINAL del formulario:

<form action="https://formsubmit.co/admin@riotripvibes.com" method="POST">
  
  <!-- Campos normales (sin cambios) -->
  <input type="text" name="name" placeholder="Nombre" required>
  <input type="email" name="email" placeholder="Email" required>
  <input type="tel" name="phone" placeholder="Teléfono">
  <select name="package" required>

    <option>Day Tour Rio - $89</option>
  </select>
  <textarea name="message" placeholder="Mensaje" required></textarea>
  
  <!-- CAMPOS OCULTOS (agregar estos) -->
  <input type="hidden" name="_next" value="https://riotripvibes.com">
  <input type="hidden" name="_subject" value="Nueva Reserva - Rio Trip Vibes">
  
  <!-- Botón (sin cambios) -->
  <button type="submit">Enviar</button>
  
</form>

IMPORTANTE:
❌ No eliminar ningún campo del form
❌ No tocar ningún JavaScript del form
✅ Solo cambiar el atributo "action"
