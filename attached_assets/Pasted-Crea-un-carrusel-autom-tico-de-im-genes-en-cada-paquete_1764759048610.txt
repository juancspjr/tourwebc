Crea un carrusel automático de imágenes en cada paquete. Las imágenes avanzan cada 4-5 segundos 
con puntos indicadores abajo. Al hacer clic en la imagen, abre los detalles del paquete.

## PARTE 1: EXPANDIR IMÁGENES EN PAQUETES

Ubicación: client/src/lib/packages.ts

Cada paquete debe tener un array "images" con 4-6 URLs de imágenes del destino.

Ejemplo:

{
id: "1",
title: "Day Tour Rio de Janeiro",
images: ["url1", "url2", "url3", "url4"],
// resto de propiedades
}
t

## PARTE 2: CREAR COMPONENTE PackageImageCarousel

Ubicación: client/src/components/PackageImageCarousel.tsx (NUEVO)

**Funcionalidad:**
- Muestra una imagen a la vez con transición fade suave (300ms)
- Puntos indicadores abajo muestran imagen actual (clickeables)
- Botones flechas izquierda/derecha (solo visible en hover)
- Avanza automáticamente cada 4-5 segundos (loop infinito)
- Pausa en hover, continúa al salir
- Al hacer clic en la imagen: ejecuta función callback para abrir detalles
- Responsive en todos los tamaños

**Props que recibe:**
- images: array de URLs
- packageTitle: nombre del paquete
- onImageClick: función callback al hacer clic en imagen

## PARTE 3: INTEGRAR EN TARJETA DE PAQUETE

Ubicación: client/src/components/PackageCard.tsx

Reemplaza la imagen estática por el carrusel:

Si paquete.images existe y tiene >1 imagen:
Renderiza PackageImageCarousel
Pasa props: images, title, callback para abrir modal
Si tiene solo 1 imagen:
Muestra imagen estática (sin carrusel)


## PARTE 4: ESTILOS CSS

Ubicación: client/src/styles/scroll-animations.css (al final)

Añade estilos para:
- Contenedor: posición relativa, overflow hidden, border radius, shadow
- Imagen: 100% ancho/alto, object-fit cover, transición fade 300ms
- Botones flechas: posición absoluta (centro vertical, izq/der), semi-transparente, hover más opaco
- Puntos: posición absoluta (abajo centro), clickeables, gris inactivo / blanco activo

## PARTE 5: LÓGICA Y INTERACTIVIDAD

El componente debe tener:
- Estado: índice de imagen actual
- useEffect: temporizador que avanza cada 4-5 segundos
- Funciones: nextImage(), previousImage(), goToImage(index)
- Eventos: onClick en imagen abre detalles, botones/puntos navegan
- Teclado: Flechas izq/der para navegación manual

## PARTE 6: LLAMADA A DETALLES

Cuando usuario hace clic en imagen:
1. Se ejecuta callback onImageClick
2. Desde PackageCard, esto abre el modal de detalles del paquete
3. El modal ya existe, solo necesita la función integrada
